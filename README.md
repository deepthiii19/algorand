# Algorand Student Learning Dashboard

## 📌 Project Description

This project is a blockchain-based student dashboard built on the Algorand network.
It allows students to store and verify their achievements securely using Algorand Standard Assets (ASAs).

## 🚀 Features

* Store student certificates on blockchain
* Verify achievements securely
* Connect wallet using Pera Wallet
* Use TestNet for safe testing

## 🛠️ Technologies Used

* Algorand Blockchain
* AlgoKit
* JavaScript SDK
* Pera Wallet

## 📂 Project Setup

1. Install AlgoKit
2. Initialize project
3. Connect to TestNet
4. Run application

## 📎 Repository Purpose

This repository will be used throughout the course to build and improve the Algorand-based application.
